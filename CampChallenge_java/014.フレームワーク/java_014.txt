Javaのフレームワーク
有名なもの、自身が興味を持つもの３つフレームワークを調べ、
特徴を調べる。
導入の方法や使用方法、特徴などオリジナルサイトに利用するつもりで調べてみる。

①Struts https://struts.apache.org/
【特徴】
　・StrutsはMVCモデルを採用したフレームワーク。
　・独自のカスタムダグの利用やActionクラスとstruts-config.xmlでの画面遷移の管理、
　　バリデータ（入力チェック）機能の提供などが挙げられる。
　
②SpringFramework https://projects.spring.io/spring-framework/
【特徴】
　・7種類のモジュールで構成
　　オブジェクトの関連付け、生成、登録などを管理する「Coreモジュール」
　　JavaBeansへのアクセス機能を提供する「Contextモジュール」
　　MVCモデルでの開発を提供する「Web MVCモジュール」等
　・Webアプリケーション以外にJavaプラットフォームで動作する
　　さまざまなアプリケーションに適用することが出来る。

③SAStruts http://sastruts.seasar.org/
【特徴】
　・設定ファイルの自動生成機能により、
　　設定ファイルの設定作業に必要な時間が大幅に軽減される。
　・ホットデプロイ機能によりアプリケーションサーバーの再起動することなく実行が
　　可能になり、スピーディなプログラミングが行える。 　
　　 (プログラムを書き換えた場合はアプリケーションサーバーの再起動が必要な場合が多い )
